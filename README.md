# EASG — Enterprise Analytics Semantic Graph Runtime

EASG is a PSA ecosystem project that provides a semantic runtime implementation.

## PSA Role

- **Role:** Runtime Provider
- **Primary Contract:** PSA Runtime Contract
- **Maturity Target:** Level 3 / PSA Native

## Mission

Implement the PSA Semantic Runtime Reference Implementation: load, manage, query, and execute PSA-compatible semantic assets.

## Responsibilities

EASG provides:

- Semantic package runtime loading
- Executable semantic context management
- Deterministic semantic query capability
- Semantic traceability and evidence generation

EASG does NOT:

- Define PSA semantic primitives or contracts
- Author domain ontologies
- Implement business application behavior

## Quick Start

Run the PSA validator:

```bash
./tools/psa-validator check .
```
